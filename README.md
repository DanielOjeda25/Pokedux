
# Pokedux 🐀

A little app of pokemon searcher using the [PokeApi](https://pokeapi.co/), dark Mode and Tailwindcss for styles.

## Technologies

 - Vite JS
 - Reactjs
 - Tailwind CSS
 - Redux Toolkit
